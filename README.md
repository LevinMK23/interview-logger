# Interview Logger

Десктопное приложение на Go для логирования интервью с временными метками.

## Функциональность

- **Редактируемое текстовое поле** - для ввода текста интервью
- **Нередактируемое текстовое поле** - для отображения лога с временными метками
- **Автоматическое логирование** - при нажатии Enter текст добавляется в лог с отметкой времени
- **Сохранение в файл** - все записи сохраняются в файл `interview.log`

## Установка и запуск

1. Убедитесь, что у вас установлен Go (версия 1.21 или выше)

2. Установите зависимости:
```bash
go mod tidy
```

3. Соберите приложение:

**Для macOS (.app bundle):**
```bash
./build.sh
```
Или вручную:
```bash
go build -o "Interview Logger" .
mkdir -p "Interview Logger.app/Contents/MacOS"
mv "Interview Logger" "Interview Logger.app/Contents/MacOS/Interview Logger"
chmod +x "Interview Logger.app/Contents/MacOS/Interview Logger"
```

**Для обычного бинарника:**
```bash
go build -o interview-logger .
./interview-logger
```

**Или запустите напрямую:**
```bash
go run .
```

## Использование

1. Введите текст в верхнее редактируемое поле
2. Нажмите Enter для добавления текста в лог
3. Текст автоматически появится в нижнем нередактируемом поле с временной меткой
4. Все записи сохраняются в файл `interview.log` в той же директории
5. Используйте кнопку "Очистить" для очистки обоих полей
6. Используйте кнопку "Сохранить лог" для закрытия приложения (лог уже сохраняется автоматически)

## Формат лога

Каждая запись в файле имеет формат (только время без даты):
```
[14:30:25] Текст интервью
```
