# Interview Logger

Десктопное приложение на Go для логирования интервью с временными метками.

## Функциональность


- **Автоматическое логирование** - пиште текст, при нажатии Enter текст добавляется в лог с отметкой времени
- **Именование файлов** - при сохранении можно указать фамилию и имя, файл будет переименован (например, `~/Иванов_Иван.log`)

## Установка и запуск

1. Убедитесь, что у вас установлен Go (версия 1.21 или выше)

2. Установите зависимости:
```bash
go mod tidy
```

3. Соберите приложение:

**Для macOS (.app bundle):**
```bash
./build.sh
```

**Для обычного бинарника:**
```bash
go build -o interview-logger .
./interview-logger
```

**Или запустите напрямую:**
```bash
go run .
```

## Использование

1. Введите текст в верхнее редактируемое поле
2. Нажмите Enter для добавления текста в лог
3. Текст автоматически появится в нижнем нередактируемом поле с временной меткой
4. Все записи автоматически сохраняются в файл `~/interview.log` (в домашней директории)
5. Используйте кнопку "Очистить" для очистки обоих полей
6. Используйте кнопку "Сохранить лог" для завершения интервью:
   - Откроется модальное окно с полем для ввода фамилии и имени
   - Введите фамилию и имя (можно нажать Enter для сохранения)
   - Файл будет переименован на основе введенного имени (например, `~/Иванов_Иван.log`)
   - Приложение закроется

## Формат лога

Каждая запись в файле имеет формат (только время без даты):
```
[14:30:25] Текст интервью
```

## Расположение файлов

- **Временный файл**: `~/interview.log` - создается автоматически при запуске
- **Сохраненные файлы**: `~/Фамилия_Имя.log` - после сохранения с указанием имени
